# Daily Email Sender

Go приложение для отправки приветственных сообщений в утро, день и вечер на указанный email.

## Установка

1. Склонируйте репозиторий или скопируйте файлы
2. Установите зависимости:
```bash
go mod download
```

## Настройка

1. Создайте файл `.env` в корневой директории:
```env
EMAIL_FROM=your-email@gmail.com
EMAIL_PASSWORD=your-app-password
EMAIL_TO=target-email@example.com
```

### Gmail настройка:
- Включите 2FA в Google Account
- Создайте App Password в разделе Security > App Passwords
- Используйте полученный пароль (без пробелов)

## Использование

Запустите приложение:
```bash
go run main.go
```

Приложение проверяет текущее время и отправляет соответствующее сообщение:
- **До 12:00** - утреннее сообщение
- **12:00-18:00** - дневное сообщение
- **После 18:00** - вечернее сообщение

## Требования

- Go 1.21 или выше
- Google Account (для Gmail) или другой email провайдер с SMTP доступом
